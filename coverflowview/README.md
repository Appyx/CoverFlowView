# How to install?
Just add the following to your applications module gradle file.


```
repositories {
    maven {
        url 'https://dl.bintray.com/appyx/maven/'
    }
}

dependencies {
    compile 'org.appyx.android.lib.coverflowview:coverflowview:1.0.0'
}
```
